Adaptadapta un objeto de aprendizaje para una terner recursos educativos

Método de adaptación Manual

Si selecciona este método de adaptación usted puede modificar uno a uno cada elemento del objeto de aprendizaje según su criterio.

Método de adaptación Automático

Si selecciona este método de adaptación nuestro algoritmo modificara el objeto de aprendizaje de forma automática, usted puede seleccionar antes las areas que desea adaptar.

Subtitulado de videos

Permite que cada video se encuentre en un reproductor accesible y que cuenten con subtitulado para un mejor entendimiento de estos.

Los subtitulados se pueden generar de forma automática usando técnicas de inteligencia artificial o si el video está en una fuente como YouTube se podrá descargar los subtítulos de esta fuente, también puedes crear tus propios subtítulos y subirlos en un archivo str.

Barra de accesibilidad

Permite la integración de la barra de accesibilidad para una modificación de la pagina del objeto de aprendizaje, con esta barra se puede modificar el tamaño de textos, tipo de letra, contraste de fondo con respecto al texto.

Al seleccionar el botón de adaptabilidad como parte del área a optimizar la barra de adaptabilidad se incrustará en todas las paginas HTML del objeto de aprendizaje, para permitir la adaptabilidad de estas páginas mediante la configuracion de diferentes parámetros.

Párrafos de texto

Los párrafos de texto muy grandes pueden ser poco entendibles para muchos lectores, por esto no son del todo aprovechables el área de optimización de este adaptador se enfoca en párrafos grandes de texto para crear una lectura fácil mediante técnicas de inteligencia artificial y la conversión del texto a audio para un mejor entendimiento.

El área de optimización de párrafos integración de un botón para permitir la lectura fácil de este párrafo, también permite la integración de un boton para convertir el texto a un audio, este puede ser un audio generado automáticamente, grabación de voz, o un audio previamente gravado o generado que la persona disponga.

Audios

Permite la conversión de audio a una formal visual, es decir en texto esto para facilitar el entendimiento de los recursos auditivos mediante la lectura.

Imágenes

Permite agregar la descripción de las imágenes del objeto de aprendizaje, esto con la finalidad de que todas las imágenes se encuentren con una descripción siguiendo los estándares HTML para la accesibilidad.
