Adaptación de Párrafos de Texto

En esta sección veremos como adaptar un párrafo de texto para poner un audio o un párrafo alternativo.

Crear párrafo de lectura alternativa

Para crear un párrafo de lectura alternativa puedes dar click en el botón de editar en la primera sección de lectura fácil puedes editar tu párrafo que se mostrara de forma alternativa al párrafo original en este párrafo alternativo opta por usar una lectura fácil para que sea mucho más comprensible el párrafo original.

Crear audio alternativo

También puedes mostrar un audio de forma alternativa para el párrafo, para crear un audio puedes dar click en editar y en la sección Audio, tienes diferentes alternativas para crear un audio, puedes subir un audio, puedes gravar un audio o puedes generar de forma automática, al crear un audio se mostrará en el objeto de aprendizaje de forma alternativa para escuchar el párrafo de texto.
