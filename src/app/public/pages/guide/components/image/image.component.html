Adaptación de Imágenes

En esta sección veremos como adaptar imágenes para crear diferentes previsualizaciones de esta.

Editar descripción

La parte fundamental de una imagen es su descripción, esta descripción ayuda se muestra cuando la imagen no este disponible o el lector de pantalla lee esta descripción para leer de que se trata la imagen, esta descripción se inserta en el ALT de la imagen, para editar esta descripción solo da click en el botón de Editar descripción y podrás editar la descripción acorde a lo que se trata la imagen.

Previsualización de imagen

Para agregar una previsualización de pantalla completa a la imagen en el objeto de aprendizaje pudes marcar el check de previsualización de imagen, al marcar o desmarcar se agregará o se quitará la previsualización en pantalla completa de la imagen en el objeto de aprendizaje.

Crear tablas

Una opción interesante que tienes disponible es el poder crear una tabla, si tu imagen es una tabla de datos, esta opción te permite crear una tabla de forma manual, podrás agregar las filas y columnas que desees para crear tu tabla.
