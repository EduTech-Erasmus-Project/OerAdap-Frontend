Área de trabajo

En esta área encontraras las diferentes secciones para adaptar el objeto de aprendizaje.

Par la vista previa del objeto de aprendizaje puedes acceder a la sección de Objeto de aprendizaje adaptado y objeto de aprendizaje original.

Para facilitar la edición de las áreas puedes acceder a la sección Todos los audios, Todas las imágenes y Todos los videos, este apartado se listará todos los recursos de todas las páginas del objeto de aprendizaje, para ahorrar tiempo en la búsqueda de cada uno de estos recursos.

También puedes buscar el contenido según la pagina del objeto de aprendizaje, este contendió lo puedes ubicar situándote en la sección Objeto de aprendizaje adaptado y seleccionando la página que desees buscar el contenido.
